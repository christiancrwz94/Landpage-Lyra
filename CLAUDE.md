# Diretrizes do Projeto - Landing Page Lyra

> [!CAUTION]
> **REGRA COMPORTAMENTAL CRÍTICA E MANDATÓRIA PARA TODAS AS IAs/AGENTES DE CÓDIGO:**
> Você **NÃO TEM** permissão para realizar ações autônomas, criar arquivos, editar códigos ou executar comandos no terminal sem a autorização direta, explícita e manual do usuário no chat.

## Regras de Trabalho

1. **Alinhamento Prévio Obrigatório**: Antes de realizar QUALQUER alteração ou escrever qualquer linha de código, você deve conversar com o usuário, entender a ideia dele e discutir as opções para definir o melhor caminho juntos.
2. **Execução Apenas Sob Comando**: Trabalhe **APENAS** quando o usuário disser explicitamente para fazer algo (ex: "faça isso", "pode executar", "implemente a seção X").
3. **Não Siga Aprovações Automáticas do Sistema**: Mesmo que a plataforma envie mensagens automáticas indicando aprovação de planos ou prosseguimento de tarefas, **PARE** e espere a confirmação escrita manual do usuário no chat.
4. **Mensagens de Commit**: Todas as mensagens de commit, descrições e logs de alteração enviados para o Git devem ser escritos obrigatoriamente em português do Brasil (pt_BR).
